# 🏥 Amartya — AI Health Assistant

Amartya is a sleek, responsive, ChatGPT-inspired web application frontend engineered specifically for health-related consultations. Designed with a clean medical aesthetic, the platform focuses on delivering structured, clear, and highly readable healthcare guidance, lifestyle tips, and symptom overviews.

Live Demo: `[Insert Live Deployment Link Here - e.g., GitHub Pages/Vercel]`

---

## ✨ Features

- **💬 ChatGPT-Style Conversational UI:** Beautifully styled, highly readable chat bubbles designed for complex medical layouts, bullet points, and warning boxes.
- **📊 Health Stats Dashboard:** A quick-glance tracking widget displaying daily vitals like Hydration levels, Steps walked, and Heart Rate (BPM).
- **🗂️ Interactive Sidebar:** Full navigation suite featuring a "New Consultation" button, conversation history logs with delete capabilities, and an active search filter bar.
- **🚨 Safety & Alert Systems:** Prominent emergency banners and persistent medical disclaimer footers prioritizing user safety.
- **⚙️ Health Profile & Settings Modals:** Dedicated user profile pop-ups to securely log age, blood type, medical history, and allergies.
- **🌓 Modern UI/UX Elements:** Fully responsive top bar equipped with active notification badges, theme toggles, and interactive suggestion cards to autofill queries.

---

## 🛠️ Tech Stack

- **Markup & Structure:** HTML5
- **Styling & Layout:** CSS3 (Custom Flexbox, CSS Grid, and responsive breakpoints)
- **Icons & Typography:** FontAwesome v6.5.0 & Google Fonts (Inter)
- **Interactions:** Vanilla JavaScript (for modal triggers, sidebar toggles, and chat behaviors)
